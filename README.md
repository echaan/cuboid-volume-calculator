# Volume Calculator App

A simple Android application used to calculate the volume of a rectangular prism (cuboid).
Users can input the length, width, and height, and the application will compute the volume based on those values.

---
## Technology Stack

* Kotlin
* Android Studio (Empty View Activity)
* XML-based UI (View system)
* AppCompat Activity

---

## How It Works

1. User enters three values: length, width, and height
2. User taps the Calculate button
3. The app computes the volume using the formula:

```
volume = length × width × height
```

4. The result is displayed on the screen
